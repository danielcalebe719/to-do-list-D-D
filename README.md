# To do List D&D
## Explicação da aplicação
Esse site faz o registro de tarefas, com descrição, título e status que pode ser concluida ou pendente.
A ideia é registrar atividades a serem feitas e ter um controle sobre a rotina.

## Testes
As pastas /cypress e /playwright contem arquivos que testam todas as funcionalidades do programa.

## Configurar ambiente de teste

### Cypress

Para testar a aplicação com Cypress, siga os passos:
Instale o Cypress via npm:


        npm install cypress 

        
Inicie o Cypress:


        npx cypress open

        
Arraste os arquivos de /cypress no repositorio para /cypress/e2e


### Playwright

Para testar a aplicação com Playwright, siga os passos:
Instale o Playwright via npm:


        npm install playwright

        
Solte os arquivos de /playwright no repositorios para /teste/ no codigo
